UhlemannLab Pipeline Toolset

Containerized Snakemake workflows for bacterial WGS, RNA-seq, and ONT/TnSeq tasks.

Snakemake Docker Micromamba Multi-env Wrapper Scripts
ONT WGS

longWGS

Assembly, polishing, QC, coverage, and annotation for bacterial long reads.

Entrypoint: longWGS/Go_longWGS_V1_1.sh

Illumina WGS

shortWGS

fastp, Kraken2, SRST2 (MLST/ARG/Plasmid), TETyper, and summary report generation.

Entrypoint: shortWGS/Go_shortWGS.sh

Illumina WGS

PFsnake

P. falciparum workflow with variant, CNV, mixed infection, and drug resistance outputs.

Entrypoint: PFsnake/Go_PFsnake.sh

RNA-seq

RNake

Bacterial RNA-seq processing and count-table generation for downstream analysis.

Entrypoint: RNake/Go_Rnake.sh

ONT / TnSeq

TnSeq_ONT

Anchor-based flank extraction and clustering for ONT amplicon/TnSeq-style analysis.

Entrypoint: TnSeq_ONT/Go_search_tnseq_flank.py

Source repository root: UhlemannLab. For usage details, see each pipeline README.
