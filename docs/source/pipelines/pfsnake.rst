PFsnake
=======

Focus
-----

Illumina *P. falciparum* workflow for mapping, variant calling, CNV, mixed infection, recombination, and drug resistance summaries.

Entrypoint
----------

``PFsnake/Go_PFsnake.sh``

Key runtime options
-------------------

- ``-p 1|2`` for single-end/paired-end mode
- ``-n`` dry-run
- ``-K`` keep-going
- ``-P 0|1`` progress monitor toggle
