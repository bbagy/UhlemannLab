ONT WGS

longWGS

Bacterial ONT workflow for assembly, polishing, QC, coverage and annotation.

Build

cd longWGS
docker build -t longwgs .

Run

./longWGS/Go_longWGS_V1_1.sh \
  -i /path/to/fastq \
  -o /path/to/output \
  -d /path/to/db \
  -K

Options

FlagDescription
-nDry-run
-KKeep-going
-p 0|1Porechop off/on
-sCustom snakefile directory
