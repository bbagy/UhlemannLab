longWGS

Bacterial ONT workflow for assembly, polishing, QC, coverage and annotation.

Status: this pipeline is tested in Nook and Core3 workstation.

Build

cd longWGS
docker build -t longwgs .

Run

./longWGS/Go_longWGS_V1_1.sh \
  -i /path/to/fastq \
  -o /path/to/output \
  -d /path/to/db \
  -K

Pipeline Flow

Prefilter→ QC / Porechop→ Autocycler→ Medaka→ QUAST + CheckM2→ Coverage→ Bakta

Options

FlagDescription
-nDry-run
-KKeep-going
-p 0|1Porechop off/on
-sCustom snakefile directory

Expected Input

FASTQ_DIR/
  sample1.fastq.gz
  sample2.fastq.gz
DB/
  medaka_models/
  plassembler_db/
  (other rule-required DBs)

Main Outputs

1_QC/ 2_quast/ 3_autocycler/ 4_medaka/
5_checkm2/ 6_coverage/ 7_bakta/
8_Bandage_image/ (optional)
