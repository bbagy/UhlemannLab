longWGS

ONT bacterial WGS assembly/polishing/QC/annotation pipeline.

Build

cd longWGS
docker build -t longwgs .

Run

./longWGS/Go_longWGS_V1_1.sh \
  -i /path/to/fastq \
  -o /path/to/output \
  -d /path/to/db \
  -K

Useful Options

-n  dry-run
-K  keep-going
-p  0|1 (porechop off/on)
-s  custom Snakefile directory
