Illumina WGS

shortWGS

QC, Kraken2 species inference, SRST2 typing, TETyper, and HTML/CSV summary generation.

Build

cd shortWGS
docker build --network=host -t shortwgs:1.0 .

Run

./shortWGS/Go_shortWGS.sh \
  -i /path/to/fastq \
  -o /path/to/output \
  -d /path/to/WGS_DB2 \
  -k /path/to/kraken2_db \
  -r /path/to/GoWGS \
  -K

Options

FlagDescription
-nDry-run
-KKeep-going
-cSnakemake cores
-mCustom docker image tag

Expected Input

FASTQ: paired-end (*.fastq.gz)
WGS_DB2:
  CARD/ PlasmidFinder/ MLST/ tetyper/
Kraken2 DB directory
GoWGS host directory (for Rmd template path)

Main Outputs

1_fastp_out/ 2a_kraken2/ 2_ST_srst2_out/
3_ARGs_srst2_out/ 4_Plasmid_srst2_out/ 5_TETyper/
multiqc_report.html
summary_master.csv
summary_report.html
