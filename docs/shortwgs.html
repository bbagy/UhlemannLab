shortWGS

Illumina bacterial typing workflow (Kraken2, SRST2, TETyper, summary report).

Build

cd shortWGS
docker build --network=host -t shortwgs:1.0 .

Run

./shortWGS/Go_shortWGS.sh \
  -i /path/to/fastq \
  -o /path/to/output \
  -d /path/to/WGS_DB2 \
  -k /path/to/kraken2_db \
  -r /path/to/GoWGS \
  -K

Useful Options

-n  dry-run
-K  keep-going
-c  cores
-m  custom docker image tag
