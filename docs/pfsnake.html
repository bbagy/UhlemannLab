PFsnake

P. falciparum pipeline for variant calling, CNV and drug-resistance summaries.

STATUS: TEST IN PROGRESS

Pipeline Flow

Prefilter→ fastp→ BWA / BAM QC→ GATK / Variants→ CNV→ Mixed Infection→ Drug Resistance

Build

cd PFsnake
docker build -t pf-snake:1.0 .

Run

./PFsnake/Go_PFsnake.sh \
  -i /path/to/fastq \
  -o /path/to/output \
  -d /path/to/ref.fasta \
  -p 2 \
  -K

Real Example

Go_PFsnake.sh \
  -i 1_fastqs \
  -o 2_pfsnake_out \
  -d /media/uhlemann/Core3_V2/REF/pf_ref.fasta \
  -p 2 \
  -K

Options

FlagDescription
-nDry-run
-KKeep-going
-p 1|2Single-end / paired-end mode
-cSnakemake cores

Expected Input

FASTQ directory
Reference FASTA
Optional GFF

Main Outputs

1_QC/ 2_bwa/ 3_bam_qc/ 4_gatk_out/
6_cnv/ 7_mixed_infection/ 8_recomb_explore/
9_drug_resistance/
