RNA-seq

RNake

Bacterial RNA-seq trimming, mapping and count-table generation workflow.

Build

cd RNake
docker build -t rnake:latest .

Run

./RNake/Go_Rnake.sh \
  -i /path/to/reads \
  -o project_name \
  -g /path/to/genome.fna \
  -a /path/to/annotation.gff \
  -K
